# LeakedIn

# Installation

```bash
# Clone the repository
git clone git@github.com:davidmoremad/LeakedIn.git

# Install requirements
pip install -r requirements.txt

# Set up the environment variables (.env file optional)
export PHANTOMBUSTER_API_KEY="your_phantombuster_api_key"

# Run the application
python main.py -u "https://www.linkedin.com/in/username"
```
